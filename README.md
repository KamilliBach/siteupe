# site-upe
Este é o repositório do projeto do Site Oficial da União Paranaense dos Estudantes (UPE), desenvolvido como um MVP para modernizar a comunicação, mobilização e transparência da entidade.
#  Site Oficial da UPE - União Paranaense dos Estudantes (MVP)

## Objetivos Principais do MVP

1.  **Comunicação Imediata:** Exibir notícias, artigos e pautas de luta de forma clara.
2.  **Mobilização:** Dar destaque aos próximos eventos, atos e como participar.
3.  **Transparência:** Apresentar a capilaridade da diretoria no estado.
